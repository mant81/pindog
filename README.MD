PinDog(핀도그)는 현재 위치를 빠르게 PIN하고,
저장·공유·QR 코드로 간편하게 전달할 수 있는 위치 저장 앱입니다.

• GPS 기반 정확한 현재 위치 감지
• 지도 위에 PIN 표시 및 저장
• 저장된 위치 리스트 관리
• 주소 복사 및 지도 앱으로 바로 이동
• 텍스트 공유 + QR 코드 생성 공유
• 즐겨찾기 위치 빠른 공유
• 가볍고 빠른 앱 실행

PinDog는 복잡한 설정 없이
현재 위치를 한 번의 터치로 저장하고
필요할 때 빠르게 공유할 수 있습니다.

지도 링크, 주소, QR 코드까지
상황에 맞게 자유롭게 공유하세요.
